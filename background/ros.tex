\section{ROS Attacks}
\label{background:ros}

Unlike prior multi-party Schnorr signature schemes in the literature~\cite{DrijversEFKLNS19,BenhamoudaLLOR20},
both FROST in Chapter~\ref{chp:frost} and Arctic in Chapter~\ref{chp:arctic} are secure against concurrent adversaries which may attempt ROS-style attacks.
%
The ROS (Random Inhomogeneities in a Overdetermined Solvable System of Linear Equations) problem~\cite{schnorr06,BenhamoudaLLOR20,Schnorr01,FuchsbauerPS20}
over $\Zq$ in dimension $\ell$
is defined as follows.
At a high level,
the goal is to find a solution to $\ell+1$ linear equations modulo $q$,
where some elements in the equations are output from a random oracle.
In other words,
given a random oracle $\HROS$ and a set of arbitrary functions
$f_j, j \in \{ j, \ldots, f_{\ell+1} \}$,
where $f_j(\Zq^\ell) \rightarrow \Zq$,
the goal is to find coefficients $a_{i} \in \Zq, i \in \set{\ell}$ such that
\[
  a_{1} c_1 + a_{2} c_2 + \ldots + a_{\ell} c_\ell = \HROS(f_{\ell+1}((a_{i})_{i \in \set{\ell}}, \aux))
\]

\noindent where $c_j = \HROS(f_j((a_{i})_{i \in \set{\ell}}, \aux))$.
It is always possible to find the first $\ell-1$ values simply by setting
$a_{i} = 1, i \in \set{\ell-1}$;
the challenge is to find the $\ell\supth$ solution.

Wagner~\cite{Wagner02} and Benhamouda et al.~\cite{BenhamoudaLLOR20} showed how to solve the ROS problem in subexponential time,
and demonstrated its applicability to the forgeability in a range of blind and multi-party
signature schemes---including blind and threshold Schnorr signatures---assuming a concurrent adversary,
which may open an arbitrary number of signing sessions simultaneously.
The proofs for prior multi-party Schnorr signature schemes shown to be insecure by
Drijvers et al.~\cite{DrijversEFKLNS19} and Benhamouda et al.~\cite{BenhamoudaLLOR20} either considered only a sequential adversary which opened only one signing session at a time,
or overlooked subtle errors in the proof.
